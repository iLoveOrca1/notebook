# Burp Suite Shortcuts

# Navigation

| Shortcut     | Action             |
|--------------|--------------------|
| Ctrl+Shift+P | go to proxy tab    |
| Ctrl+Shift+R | go to repeater tab |
| Ctrl+Shift+I | go to intruder tab |

# Intercept
| Shortcut | Action                       |
|----------|------------------------------|
| Ctrl+f   | forward intercepted requests |

# HTTP History
| Shortcut | Action           |
|----------|------------------|
| Ctrl+r   | move to repeater |
| Ctrl+i   | move to intruder |
